## Overview

Despite the classical CLI usage, you can also use it in scripts like:

```npm
npm install create-docs-app
```

```ts
import { create } from 'create-docs-app';

await create({
  outputDir: 'my-app',
  template: '+next+-docs-mdx',
  packageManager: 'pnpm',
});
```
